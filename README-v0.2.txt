# SWEP v0.2 update

To jest aktualizacja dla wcześniejszej wersji HTML/JS.

## Jak wgrać
1. Rozpakuj ZIP.
2. Podmień w repozytorium pliki:
   - index.html
   - css/style.css
   - js/app.js
3. Nie nadpisuj swojego `js/config.js`, jeśli masz już wklejony działający klucz Supabase.
4. Commit → Push.
5. Poczekaj na Cloudflare i odśwież stronę Ctrl+F5.

## Co dodano
- Nowy wygląd pulpitu.
- Liczniki dokumentów, kart pojazdów i oczekujących spraw.
- Lepszy panel kart pojazdów.
- Wyszukiwarka kart pojazdów.
- Panel dokumentów dowódcy z filtrem statusów.
- Lepszy układ mobilny.
