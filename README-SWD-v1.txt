# SWD v1.0 update

Aktualizacja zmienia nazwę systemu z SWEP na SWD.

## Jak wgrać
1. Rozpakuj ZIP.
2. Podmień w repozytorium:
   - index.html
   - css/style.css
   - js/app.js
3. Nie podmieniaj `js/config.js`, bo tam masz działający klucz Supabase.
4. Zrób Commit i Push.
5. Po wdrożeniu odśwież stronę Ctrl+F5.

## Zmiany
- SWEP zmienione na SWD.
- Nowa nazwa: System Wspomagania Dowodzenia.
- Rozbudowane menu: pojazdy, dokumenty, panel dowódcy, funkcjonariusze, służby, ogłoszenia, statystyki.
- Nowy pulpit z profilem funkcjonariusza.
